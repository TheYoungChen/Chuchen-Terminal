## 说明 / Notes

- Topics 会出现在仓库首页，用于 GitHub 搜索、分类和相关项目发现。
- Description 是最容易被搜索结果和个人主页 pinned card 看见的一句话。
- Website 不是必填项，不相关的网站会降低仓库信息一致性。

- Topics appear on the repository page and help GitHub search, classification, and discovery.
- Description is the main sentence shown in search results and pinned repository cards.
- Website is optional. An unrelated website weakens the project signal.
