# Code of Conduct

## Our Pledge

We aim to keep Chuchen-Terminal a respectful and practical space for discussing local development workflows, terminal tooling, and desktop app improvements.

## Expected Behavior

- Be respectful and direct.
- Focus feedback on the code, design, or documentation.
- Provide clear reproduction steps for bugs.
- Avoid posting private credentials, private logs, or other sensitive information.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing another person's private information.
- Posting secrets, tokens, private logs, or private workspace data.
- Repeated off-topic disruption.

## Reporting

If you need to report unacceptable behavior, open a private security or maintainer contact channel when available. If no private channel is available, avoid sharing sensitive details publicly.
